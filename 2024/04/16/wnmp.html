nginx-1.24.0.zip解压, 复制文件到文件夹(e.g. C:\iu\nginx)

启动: 1) 双击 nginx.exe 启动 2) 命令行执行 start nginx.exe 启动 3)添加到系统服务启动

3)PHP

复制 php.ini-production 文件为 php.ini

启动: 1)命令行执行 php-cgi.exe -b 127.0.0.1:9000 -c php.ini 2)添加到系统服务启动[PS. 利用nssm设置为服务项时, 若参数的路径中有空格, e.g. C:\Program Files\php\php.ini, 可能会导致数据库无法使用！！！]

验证: [重启 nginx], 新建文件 info.php 文件, 浏览器访问 localhost/info.php or 127.0.0.1/info.php, 出现如下页面即配置成功! (前提: nginx.conf 配置文件已修改, 支持 php 解析)

验证: 浏览器访问 localhost/phpmyadmin, 出现如下页面即配置成功!

3. 注册 nginx 和 php-cgi 为系统服务项

1)NSSM下载

访问 NSSM官网 下载NSSM

以下链接可直接下载[最终版] nssm 2.24.zip (2014-08-31)

下载完成后解压得到相关文件, 根据系统选择32位(win32)或 64位(win64)

2)nginx

将 win64 文件夹下的 nssm.exe 文件复制到 nginx 根目录下

管理员权限运行 cmd 进到 nginx 的根目录下, 输入 nssm install nginx

Path: 选择 nginx.exe 文件

Startup directory: nginx.exe文件所在的目录路径[默认选项即可!]

Arguments: 此处参数为空

以上选择完成后, 点击 Install service 将 nginx 配置为服务项, 名称 nginx

3)php-cgi

将 win64 文件夹下的 nssm.exe 复制到 php 根目录下

管理员权限运行cmd进到php的根目录下, 输入nssm install php

Path: 选择 php-cgi.exe 文件

Startup directory: php-cgi.exe文件所在的目录路径[默认选项即可!]

Arguments: -b 127.0.0.1:9000 -c C:iu\php\php.ini

以上选择完成后, 点击 Install service 将 php-cgi 配置为服务项, 名称 php

4)设置开机自启动

Win+R 输入 services.msc 进入服务设置

分别找到 nginx, php 服务项, 将启动方式设置为 自动
